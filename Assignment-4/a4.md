## Task 1: Image Segmentation using K-Means Clustering

### Requirements
- Convert input images into an appropriate feature space (e.g., RGB or Lab color space).
- Apply K-means clustering with K = 4.
- Use 2 sets of different random initializations (seeds) and analyze variation in results.
- Assign each pixel to its nearest cluster center.
- Generate segmented output images for each initialization.

### Outputs
- Original image
- Segmented image (K = 4)
- Comparison of results from 2 sets of different random seeds

### Analysis
- Discuss how initialization affects segmentation quality.
- Comment on the limitations of K-means for image segmentation.

## Task 2: Image Segmentation using a Neural Network

You may choose one of the following architectures:
- Fully Convolutional Network (FCN)
- U-Net
- SegNet
- Any lightweight segmentation model

### Requirements
- Train a neural network model for segmentation (use ground truth masks provided in the dataset).
- Test the model on the provided testing dataset.

### Outputs
- Predicted segmentation masks (result)
- Overlay of segmentation on original images
- Comparison of the predicted segmentation masks with the provided ground truth masks

### Analysis
- Describe model architecture briefly.
- Report performance metrics (compared with ground truth info).
- Discuss strengths and weaknesses of the neural network compared to K-means.
